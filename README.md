### Installation

#### Clone repo

``` bash
# clone the repo
$ git clone https://github.com/Huxteen/mini-crm.git

# go into app's directory
$ cd mini-crm

# install app's dependencies
$ composer install

# install app's dependencies
$ npm install

```
#### Usage

# go to ui/vue-paper directory
$ cd ui/vue-paper

# install app's dependencies
$ npm install

# back to mini-crm root directory
$ cd ../../

# generate mixing
$ npm run dev

# and repeat generate mixing
$ npm run watch

# start local server
$ php artisan serve
